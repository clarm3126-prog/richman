📋 발급 방법:
1. 새 Fine-grained Token 발급 페이지 열기
2. Token name: richman alerts
3. Repository access → "Only select repositories" → richman 선택
4. Repository permissions 펼치기:
  • Contents → Read and write (필수)
  • Actions → Read and write (선택, 즉시 알림용 ✨)
5. 하단 Generate token → 생성된 토큰 복사 → 위 입력란에 붙여넣기

⚠️ Token은 본인 핸드폰/PC에만 저장됩니다 (서버 X). 분실 우려 있으면 GitHub에서 언제든 revoke 가능.
